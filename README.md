# Bank Customer Default Analysis

## Project Overview

This project focuses on analyzing customer credit behavior and identifying factors associated with credit card default risk using banking customer data.

The project follows a structured data science workflow, starting with data preprocessing, feature engineering, and exploratory data analysis (EDA) before moving to machine learning model development.

---

## Business Objective

Financial institutions need to identify customers who are likely to default on their credit obligations.

The objective of this project is to:

* Improve data quality through preprocessing
* Create meaningful business-driven features
* Analyze customer behavior patterns
* Identify key factors influencing default risk
* Prepare data for predictive modeling

### Phase 2: Exploratory Data Analysis (EDA) ✅

Analysis performed:

* Target variable analysis
* Age distribution analysis

* Credit score analysis
* Default vs Credit Score analysis
* Default vs Income analysis
* Correlation analysis
* Customer risk segmentation

---

## Key Insights

* Overall default rate: 8.16%
* Most customers belong to the Good Risk category
* Previous defaults are strongly associated with future default behavior
* Higher credit utilization is linked to higher default risk
* Customers with lower credit scores are more likely to default
* Credit score appears to be one of the strongest predictors of default risk
